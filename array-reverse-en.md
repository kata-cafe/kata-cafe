# Array Reverse

Create `arrayReverse` function to reverse order of items in an array.
Cannot use `reverse` built in function from any language or library.

## Test Cases

```js
  arrayReverse([1, 2, 3, 4]) -->  [4, 3, 2, 1]
```

```js
  arrayReverse(['a', 'c', 'b']) -->  ['b', 'c', 'a']
```
