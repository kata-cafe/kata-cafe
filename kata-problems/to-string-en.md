# toString

Create `toString()` function without using `toString` built in function from any language or library.

## Test Cases

```js
  toString(123) -->  "123"
```

```js
  toString(Math.PI) -->  "3.141592653589793"
```

```js
  toString(true) -->  "true"
```

```js
  toString([2, 4, 8, 17]) --> "[2, 4, 8, 17]"
```
